### shared/tokens/

#### POPUP_DATA
Defines a custom Angular dependency injection token used to pass dynamic data to popup components.  
The token allows dynamically created components to receive external data without relying on direct component inputs.  
[POPUP_DATA](https://github.com/valeriinikolaichuk/powerlifting_competition_platform/blob/main/frontend/src/app/shared/tokens/popup-data.token.ts) creates a communication layer between the popup infrastructure and dynamically rendered content.
